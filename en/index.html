About

🛡️ About the Project

SentinelAU is being developed with a strong focus on lobby protection, host convenience, and a more stable gameplay experience.

Main Focus

  • lobby control
  • host convenience
  • protection against current threats
  • gameplay stability

Project Purpose

SentinelAU is being built as an anti-cheat mod that combines lobby protection, host management tools, and additional functionality designed to improve convenience and overall control during gameplay.

Features

⚙️ Current Features

At this stage, the project already includes core protection systems, host tools, and elements of its own custom interface.

🛡️

Protection

  • protection against current exploits
  • a protection system that drops cheater attacks outside the host, currently not yet fully implemented
⚙️

Host Tools

  • auto-host functionality through the mod menu
  • viewing and editing the banlist and whitelist through the mod menu
  • nickname-based banning
🎨

Additional Functionality

  • color locking by friend code through a file
  • a licensing system requiring an activation key after installation, otherwise the anti-cheat will not work
  • custom design and visual style
  • and much more
