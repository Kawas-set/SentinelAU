Project purpose

SentinelAU is being created as an anti-cheat mod that combines lobby protection, host management tools, and extra functionality that improves convenience and control during gameplay.

Why SentinelAU

✨ Why SentinelAU

The project is designed not as a random set of disconnected features, but as a cohesive system for hosts, where protection, control, and convenience work together.

🎯

Host-first approach

SentinelAU is built to give the host more control over the lobby instead of forcing them into constant manual cleanup every match.

🧱

Protection as the foundation

At the core of the project is lobby protection and gameplay resilience, while additional features are meant to support the mod rather than distract it from its main purpose.

🎛️

Practical feature set

The mod’s tools are selected with real use cases in mind: useful management, access to the right information, and fewer unnecessary actions in the game itself.

Features

⚙️ Current features

At the moment, the project already includes core protection mechanisms, host tools, and elements of its own custom interface.

🛡️

Protection

  • protection against current exploits
  • implemented protection that drops attacks from cheaters outside the host, although it is not yet complete
⚙️

Host tools

  • auto-host function through the mod menu
  • viewing and editing the ban list and whitelist through the mod menu
  • ability to ban by nickname
🎨

Additional functionality

  • color binding by friend code through a file
  • implemented licensing system: after installing the mod, an activation key is required or the anti-cheat will not work
  • custom design and visual style
  • and much more
