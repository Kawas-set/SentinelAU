# SentinelAU 

**SentinelAU** — мод для **Among Us**, созданный для защиты лобби, удобного управления и более стабильной игры.

## О проекте
SentinelAU разрабатывается с упором на контроль лобби, удобство для хоста и защиту от актуальных угроз.

## На данный момент мод включает в себя
- защиту от актуальных эксплоитов
- реализована защита, которая дропает атаки читера вне хоста, но на данный момент не в полном объёме
- функцию автохоста через меню мода
- просмотр и редактирование бан-листа и бел-листа через меню мода
- возможность бана по нику
- закрепление цвета по friend code через файл
- реализована система лицензирования: после установки мода требуется ключ активации, без которого античит не будет работать
- собственный дизайн и визуальное оформление
- и многое другое
