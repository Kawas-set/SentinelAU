**SentinelAU** — мод для **Among Us**, созданный для защиты лобби, удобного управления и более стабильной игры.

## 🛡️ О проекте
SentinelAU разрабатывается с упором на:
- контроль лобби
- удобство для хоста
- защиту от актуальных угроз
- стабильность игрового процесса

## ⚙️ Текущие возможности
- защита от актуальных эксплоитов
- реализованная защита, которая дропает атаки читера вне хоста, но на данный момент не в полном объёме
- функция автохоста через меню мода
- просмотр и редактирование бан-листа и бел-листа через меню мода
- возможность бана по нику
- закрепление цвета по friend code через файл
- реализованная система лицензирования: после установки мода требуется ключ активации, без которого античит не будет работать
- собственный дизайн и визуальное оформление
- и многое другое
