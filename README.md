<p align="center"><b>🔰 SentinelAU</b> — античит-мод для <b>Among Us</b>, разработанный для защиты лобби, расширения возможностей хоста и создания более удобного и стабильного игрового процесса.</p>

---

## 📌 Статус проекта

SentinelAU находится в активной разработке.

**Текущие приоритеты:**
- расширение систем защиты
- развитие инструментов для хоста
- доработка интерфейса
- подготовка к первому публичному релизу

## 🛡️ О проекте

SentinelAU разрабатывается с упором на:
- контроль лобби
- удобство для хоста
- защиту от актуальных угроз
- стабильность игрового процесса

## ⚙️ Текущие возможности

- защита от актуальных эксплоитов
- реализованная защита, которая дропает атаки читера вне хоста, но на данный момент не в полном объёме
- функция автохоста через меню мода
- просмотр и редактирование бан-листа и бел-листа через меню мода
- возможность бана по нику
- закрепление цвета по friend code через файл
- реализованная система лицензирования: после установки мода требуется ключ активации, без которого античит не будет работать
- собственный дизайн и визуальное оформление
- и многое другое

## 🚀 Планы

В планах — дальнейшее расширение и развитие функционала мода.

**Ориентировочные сроки релиза:** лето-осень 2026 года.

## ℹ️ О репозитории

Данный GitHub-репозиторий используется как официальная публичная страница проекта **SentinelAU**.

Здесь размещается общая информация о моде, описание его возможностей, визуальные материалы, статус разработки и другие публичные сведения о проекте.

Полный исходный код проекта не публикуется и публиковаться не планируется.
