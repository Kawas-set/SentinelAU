About

🛡️ О проекте

SentinelAU разрабатывается с упором на защиту лобби, удобство для хоста и более стабильный игровой процесс.

Основной фокус

  • контроль лобби
  • удобство для хоста
  • защита от актуальных угроз
  • стабильность игрового процесса

Назначение проекта

SentinelAU создаётся как античит-мод, который сочетает в себе защиту лобби, инструменты управления для хоста и дополнительный функционал, повышающий удобство и контроль во время игры.

Features

⚙️ Текущие возможности

На данный момент проект уже включает в себя базовые защитные механизмы, инструменты хоста и элементы собственного интерфейса.

🛡️

Защита

  • защита от актуальных эксплоитов
  • реализованная защита, которая дропает атаки читера вне хоста, но на данный момент не в полном объёме
⚙️

Инструменты хоста

  • функция автохоста через меню мода
  • просмотр и редактирование бан-листа и бел-листа через меню мода
  • возможность бана по нику
🎨

Дополнительный функционал

  • закрепление цвета по friend code через файл
  • реализованная система лицензирования: после установки мода требуется ключ активации, без которого античит не будет работать
  • собственный дизайн и визуальное оформление
  • и многое другое
